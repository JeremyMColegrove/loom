# How To Read This Book

Use the book in two passes.

## Pass 1: Learn the model

Read chapters in order:
1. Overview
2. Program Structure
3. Getting Started
4. Security Policy
5. Syntax and Directive Reference

Goal of pass 1: understand how Loom thinks about work.

## Pass 2: Use as reference

After first pass, jump by task:
- Setting up policy: Chapter 4.
- Writing and debugging scripts: Chapter 5.

## Practical reading tip

Keep a sample script open while reading.

For each section, ask:
- What does this feature do?
- When should I use it?
- What failure mode does it prevent?
