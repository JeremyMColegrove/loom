# Chapter 1: Overview

This chapter gives the big picture for Loom.

Loom was created as a lightweight scripting language for file-heavy automation work. The core problem: for complex file parsing and file routing tasks, many teams end up writing too much boilerplate in general-purpose languages.

In this chapter, you will learn:
- What Loom is and where it fits.
- Why Loom was created.
- How Loom compares to Python for this style of work.
- A first end-to-end pipeline example.

If you are new, read this entire chapter before moving on.
