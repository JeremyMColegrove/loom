# Loom Wiki Book

This folder is the source for Loom's GitHub Wiki-style docs.

The docs are written like a book:
- Chapters are grouped in numbered folders.
- Each chapter starts with a `README.md`.
- `SUMMARY.md` is the table of contents.

## Writing style

Use simple, human English:
- Short sentences.
- Real examples.
- Explain the "why" before the "how".

## Suggested flow

1. Overview
2. Program structure
3. Getting started
4. Security policy
5. Syntax and directive reference

## GitHub Wiki note

GitHub Wikis are separate repositories (`<repo>.wiki.git`).
If you want to publish this book into the wiki, copy these Markdown files into that wiki repo and keep the same folder structure.
