# Chapter 3: Getting Started

This chapter walks you from zero to your first reliable Loom run.

You will learn:
- How to set up a minimal project layout.
- How to run a first script with policy enabled.
- How to review output and verify correctness.
- How to scale from tiny tasks to larger workflows.

The goal is not just “it runs.”

The goal is “it runs safely, predictably, and is easy to maintain.”
